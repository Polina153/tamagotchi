# Тамагочи в консоли — Kotlin

Консольный тамагочи с анимацией и TUI на чистом Kotlin, без внешних зависимостей.

## Запуск готового jar (Java 8+)
```
java -jar tamagotchi.jar
```

## Сборка из исходников (IntelliJ IDEA)
1. Открыть папку как Gradle-проект
2. Run > tamagotchi.MainKt

## Управление
| Клавиша | Действие |
|---------|----------|
| ↑ ↓     | Навигация по меню |
| Enter   | Выбрать действие |
| 1       | Покормить |
| 2       | Поиграть |
| 3       | Уложить спать |
| 4       | Дать лекарство |
| 0 / Q   | Выход |

## Структура проекта
```
src/main/kotlin/tamagotchi/
├── Main.kt        — точка входа, игровой цикл, обработка ввода
├── Pet.kt         — модель питомца, статы, состояния
├── GameLogic.kt   — тики, деградация статов, действия игрока
├── Animations.kt  — ASCII-анимации по состояниям
└── Renderer.kt    — отрисовка TUI через ANSI escape-коды
```

## Архитектура
- **GameState** (object) — общее volatile-состояние между потоками
- **Поток gameTicker** — каждые 4 сек вызывает logic.tick()
- **Поток renderLoop** — каждые 500мс перерисовывает экран
- **Главный поток** — читает ввод в raw mode (stty -icanon)
- **synchronized(lock)** — защита от гонок при чтении/записи Pet

## Примечание
TUI работает в Linux/Mac терминалах. В Windows запускайте через Windows Terminal
(не cmd.exe) с поддержкой ANSI VT100.
